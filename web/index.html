Entregables

Selección del modelo
  • Selección del modelo
  • Justification: explicación del por qué se eligio el modelo en particular. Fortalezas y debilidades y por qué es adecuado para el problema dado.
  • Relevance: descripción de cómo el modelo elegido se alinea con los objetivos del proyecto y el tipo de datos disponibles. Referencias de casos de uso o estudios similares en los que este modelo haya sido eficaz.

Análisis exploratorio de datos (EDA)
  • Resumen de datos: resumen del conjunto de datos, incluido el número de observaciones, variables y tipos de datos.
  • Repositorios exógenos: identificación y descripción de cualquier fuente de datos externa que pueda integrarse para mejorar el análisis. Explicación de la relevancia y el impacto potencial de estos datos adicionales.
  • Visualización de datos: creación de visualizaciones para descubrir patrones, tendencias y valores atípicos en los datos. Uso de gráficos, cuadros y diagramas para ilustrar sus hallazgos.
  • Limpieza de datos: detalle del proceso de limpieza de datos, incluido el manejo de valores faltantes, valores atípicos e inconsistencias. Documentación de cualquier transformación o normalización aplicada.
  • Imputación de datos: explicación de los métodos utilizados para imputar los valores faltantes y justificación de la elección de las técnicas.
