Migración a BigQuery y Preparación del Modelo

Fecha de entrega: 16/10/2024

Resumen

Haciendo uso de las GCP se cargo el dataset de COVID-19 a Drive como .csv debido al gran peso (cantidad de datos). Luego, haciendo uso del enlace compartido se nos permitio publicarlo en un dataset de BigQuery. Porteriormente, creamos una Cuenta de Servicio, en Google Cloud, con una llave .json para permitir la conexión a BigQuery desde un proyecto de dbt.

Migración de datos a BigQuery

  • Extracción de información: descargamos de forma local los datos de Datos Abiertos Colombia - COVID-19 como .csv haciendo uso de la API.
  • Publicación en Drive: los datos locales se subieron a Drive como .csv, aquí
  • Inserción de los datos en un dataset de BigQuery: se creó un dataset de BigQuery y se subieron los datos de Drive con el ID `dm-09242024.raw.casos positivos covid-19`.
  • Creación de una Cuenta de Servicio: para facilitar la conexxión a BigQuery desde un proyecto de dbt, se creó una Cuenta de Servicio con una llave .json de uso personal exclusivo para el proyecto de dbt.

Finalmente, replicamos el proceso de limpieza de los datos explicados en la sección de EDA. Se puede leer más al respecto aquí.

Justificación

Dado que los datos de COVID-19 tienen una clara secuencia temporal (días consecutivos), y es crucial prever los casos futuros con base en los datos anteriores, ARIMA es una elección natural y adecuada.

  • Fortalezas: ARIMA es un modelo robusto y ampliamente utilizado en la predicción de series temporales. Es capaz de capturar patrones históricos y proyectar futuros puntos en la serie temporal con alta precisión.
  • Debilidades: requiere que los datos sean estacionarios o se transformen para alcanzar la estacionariedad. Puede no ser ideal para datos con relaciones no lineales complejas.

Relevancia del modelo

El modelo ARIMA proporciona una herramienta crítica para anticipar la evolución de pandemias, permitiendo una respuesta proactiva en la gestión de recursos sanitarios y políticas públicas.

ARIMA es especialmente relevante para analizar y predecir la propagación de enfermedades, como ha sido demostrado en estudios anteriores durante brotes epidémicos. Su capacidad para prever la evolución de los casos en el tiempo es crucial para la planificación de recursos sanitarios y la implementación de medidas de control.

Apreciaciones finales

La selección del modelo ARIMA para la predicción de la tasa de contagio de COVID-19 se basa en su capacidad para manejar series temporales con precisión. Aunque requiere datos estacionarios y puede tener limitaciones frente a relaciones no lineales, su relevancia en la previsión de brotes epidémicos lo convierte en una herramienta valiosa para apoyar decisiones en salud pública. La planificación adecuada del análisis y la evaluación rigurosa del modelo garantizarán la utilidad de las predicciones obtenidas.
